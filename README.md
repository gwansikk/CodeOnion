<h1 align="center">CodeOnion</h1>

<p align="center">빠른 생산성을 위해 기본적인 환경과 유용한 플러그인을 설정한 개발환경 템플릿입니다.</p>

- 빠른 생산성을 위해 캡슐화된 코드를 제공합니다.
- 새로운 프로젝트의 초기설정에 드는 비용을 줄일 수 있습니다.
- 코드 품질과 효율성을 빠르게 향상시킬 수 있습니다.
