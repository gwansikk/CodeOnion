## Commit Convention

| 제목     | 내용                                                                             |
| -------- | -------------------------------------------------------------------------------- |
| init     | 작업 세팅 커밋 (패키지 설치 등)                                                  |
| feat     | 새로운 기능을 추가할 경우                                                        |
| style    | 기능에 영향을 주지 않는 커밋, 코드 순서, css등의 포맷에 관한 커밋                |
| fix      | 버그를 고친 경우                                                                 |
| refactor | 코드 리팩토링                                                                    |
| docs     | 문서를 수정한 경우, 파일 삭제, 파일명 수정 등                                    |
| chore    | 빌드 테스트 업데이트, 패키지 매니저를 설정하는 경우, 주석 추가, 자잘한 문서 수정 |
